= Fonts Rails

Adds a simple helper to create the google fonts link tag.

= Install

Put this line in your Gemfile:
  gem 'fonts-rails'
  
Then bundle:
  $ bundle

= Usage
  
In your layout add a line like this in your head:
  <%= google_fonts_link_tag 'Open+Sans:300' %>
